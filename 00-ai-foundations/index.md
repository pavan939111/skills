# 00 — AI Foundations

Enduring theoretical concepts behind modern AI models — separated from `05-ai-engineering/`, which covers building production AI *systems*. This folder answers "how do these models fundamentally work," not "how do I build a product with them." Theory here changes slowly; practice in `05-ai-engineering/` changes fast.

**Status legend:** ✅ done · ⏳ pending

## Research template (every file follows this structure)

1. Definition & core concepts
2. Why it matters for building AI products (even though this is theory)
3. How it actually works (the mechanism, explained clearly)
4. Common misconceptions
5. How it constrains what's possible in `05-ai-engineering/` (e.g., context window size → context engineering; tokenization → cost and prompt design)
6. Key terminology glossary
7. Further reading / primary sources
8. AI coding-agent implementation guidelines (what an agent should know before touching model config/prompts)
