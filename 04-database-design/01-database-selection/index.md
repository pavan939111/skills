# 01 — Database Selection

Helps an AI agent determine which database (or combination of databases) is the right choice for a given use case, before any schema is designed. Each database-type file and framework file follows the 12-section research template, with special emphasis on: when to use, when NOT to use, advantages, limitations, scalability, cost, real-world examples, and common mistakes.

## Research template (every file follows this structure)

1. Definition & core concepts
2. Why it exists / what problem it solves
3. What breaks in production without it (or: what goes wrong when this database type is misapplied)
4. Best practices
5. Common mistakes / anti-patterns
6. Security considerations
7. Performance considerations
8. Scalability considerations
9. How major companies implement it (real-world examples)
10. Decision checklist (when to use / when NOT to use)
11. AI coding-agent implementation guidelines
12. Reusable checklist
