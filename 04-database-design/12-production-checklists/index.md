# 12 — Production Checklists

Quick, consolidated validation checklists to run before a database design ships to production. These files are intentionally more compact than the rest of the knowledge base — each is primarily a checklist that references back to the detailed topic files in folders 01–11, rather than a full 12-section research doc.

## Format for every file in this folder

1. Purpose (what this checklist validates and when to run it)
2. Checklist (grouped, actionable checkboxes)
3. Cross-references (links to the detailed topic files each item comes from)
4. Sign-off criteria (what "passed" looks like)
